# 📸 PhotoBooth
### *A humble space to capture and share moments.*

Welcome to **PhotoBooth**, a simple yet premium place designed to bring people together. This project was built with a clear goal: to make sharing memories feel personal, clean, and beautiful. Whether you are taking a solo photo or inviting a friend for a shared session, PhotoBooth is here to make that moment special.

**Developer:** Jemmy Francisco

---

## 🌟 About the Project
PhotoBooth is a modern web application that lets you take "film-strip" style photos right in your browser. It uses a professional **Glassmorphism** design—which means everything looks like soft, glowing glass. It is built to be fast, secure, and very easy to use on both your phone and your computer.

---

## 🚀 How to Run (Development)
If you want to run this project on your own computer, please follow these simple steps:

1.  **Download the Code**: Clone the repository or download the files.
2.  **Install Dependencies**: Open your terminal in the main folder and type:
    ```bash
    npm install
    ```
3.  **Setup Keys**:
    - Put your Firebase keys in `frontend/.env`.
    - Put your Database link in `backend/.env`.
4.  **Start the Engine**: Type this command to start both the front and back at the same time:
    ```bash
    npm run dev
    ```

---

## 🛠️ How to Build (Production)
When you are ready to show the world, you can build a small, fast version of the site:

1.  **Build Command**:
    ```bash
    npm run build
    ```
2.  **Output**: Your ready-to-use website will be in the `frontend/dist` folder.

## 📖 How to Use
Using PhotoBooth is designed to be very natural:

1.  **Join Us**: Sign in easily using your **Google Account**.
2.  **The Studio**: Click **"Open Camera"** to start taking photos. You can choose different layouts like "Strip" or "Postcard."
3.  **Socialize**: Go to the **Friends** tab to find people. Once you follow each other, you become friends!
4.  **Invite**: On your Dashboard, click on an **Online** friend to invite them to a live photo session.
5.  **Gallery**: Your last 3 prints are always waiting for you on your Dashboard to view or download.
